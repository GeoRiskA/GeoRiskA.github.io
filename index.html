Photogrammetry tools

GeSSA — GeoRiskA Scanning Software for Airphotos

GeSSA is a software that allows parallel scanning of panchromatic aerial photographs using several flatbed scanners connected to a single computer. The goal is to speed up the digitising process. The software only operates on Linux computers. The scanners must be identical.
>>> GitHub Repository

GAPPS — GeoRiskA Airphoto Pre-Processing Suite

GAPPS is a toolbox dedicated to prepare scanned aerial photographs to the photogrammetric processing. THe software allows to standardise the image size, orient the photos using tje fiducial marks, remove the margins, and optionally create a mask.
>>> GitHub Repository

Aerial Survey Planning

Collection of Python scripts used to prepare/plan an aerial imaging survey. It includes tools to select the proper camera parameters, flight speed, flight elevation, ground sampling distance, etc., based on the characteristics of the equipment used and the constraints of the flight/field survey.
>>> GitHub Repository

SfM-MVS Photogrammetry Tips

Collection of user guides, tutorials, and python/shell scripts to facilitate the use of photogrammetry to process UAV data, with a specific focus on Agisoft Metashape Pro software.
>>> GitHub Repository

Remote sensing tools

GEE-GeoTools — Google Earth Engine tools for Earth Observation of geo- and geo-hydrological hazards

GEE-GeoTools is a set of Google Earth Engine tools, managed through a Streamlit user interface, which are dedicated to the detection, mapping, and measurement of geohazards and geo-hydrological hazards. This toolbox includes the V-HOTMAP and GEO-SAR-Flood tools described here below.
>>> GitHub Repository (IN PROGRESS)

V-HOTMAP — Volcanic hotspot detection tool for Landsat-type imagery, using Google Earth Engine

V-HOTMAP is an evolution of the HOTMAP thermal hotspot detection algorithm developed by Murphy et al. (2016), for the detection of volcanic thermal anomalies on Landsat-type satellite images, in Google Earth Engine.
>>> GitHub Repository (IN PROGRESS)

GEO-SAR-Flood — Flood detection and mapping using SAR imagery and Google Earth Engine

GEO-SAR-Flood is a Google Earth Engine tool to detect and map floods using Sentinel-1 SAR amplitude images. The tool combines the change detection and backscatter intensity thresholding methods and apply them on VH-polarized amplitude images.
>>> GitHub Repository (IN PROGRESS)
